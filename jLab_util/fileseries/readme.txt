The FileSeries Toolbox for Matlab allows for recursive
operations on files (directories, deleting...), automatic
renaming and renumbering for series of files. This is
useful for handling a large number of files, for example
JPG photos.

_____________________________________________________

Installation

Download and unzip the FileSeries Toolbox into a
directory known by your Matlab installation.

_____________________________________________________

Getting started

The basics of the Davis Toolbox is to import Davis VEC
or VC7 files into a Matlab structure (or structure array)
using the function loadvec or loadset. This structure
contains the velocity fields, axes, units etc., and can
be processed by several functions of the toolbox.

Type help davis to learn more about this toolbox.
